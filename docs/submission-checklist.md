# Submission Checklist

## Required Items

- Problem statement
- Solution overview
- Prototype link (live MVP)
- Project deck
- Public GitHub repository
- Demo video

## Final Verification

- Confirm the final submitted build satisfies the organizer requirement for cloud deployment and at least one Google AI model or service.
- Open the hosted app in an incognito window.
- Register or log in successfully.
- Confirm the dashboard loads without local setup.
- Trigger one refresh cycle.
- Trigger one reroute action.
- Verify the README explains the problem, MVP scope, setup, and limitations.
- Verify the deck includes problem, solution, MVP proof, repo link, and video link.
- Watch the full video once before submitting.
